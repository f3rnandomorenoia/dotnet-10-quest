# .NET 10 Quest

Juego táctil móvil, estático y publicable en GitHub Pages para aprender .NET 10 desde cero hasta nivel experto.

## Qué incluye

- 27 misiones cortas con progresión de `Cero` a `Experto`.
- Mecánicas táctiles de elegir, multi-selección y ordenar secuencias.
- Progreso local con `localStorage`.
- Contenido basado en documentación oficial de Microsoft Learn.
- Diseño mobile-first sin backend ni dependencias de build.
